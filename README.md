# 🎵 Music Ball - 音乐可视化项目

一个交互式的3D音乐可视化应用，使用Three.js和Web Audio API实现实时音乐频率分析与球体动画效果。

## ✨ 功能特性

- 🎾 **3D球体可视化** - 使用Three.js创建的动态3D球体
- 🎵 **实时音乐分析** - 利用Web Audio API分析音乐频率
- 🌈 **动态颜色变化** - 根据音乐节奏改变球体颜色
- 📱 **响应式设计** - 完美适配各种屏幕尺寸
- 🎚️ **音乐播放器** - 支持上传和播放本地音乐文件

## 🚀 快速开始

1. 克隆仓库：`git clone https://github.com/Limohao-Netizen/11111.git`
2. 用浏览器打开 `index.html` 文件
3. 上传你的MP3音乐文件开始使用

## 🛠️ 技术栈

- **Three.js** - 3D图形库
- **Web Audio API** - 音频处理
- **HTML5 & CSS3** - 前端开发
- **Vanilla JavaScript** - 核心交互逻辑
